# School-Administration-System-Backend-APi

## Initialize Project

1. Install the required modules

   ```bash
   npm install
   ```

2. Use your MySQL Editor to create tables from `DDL.sql` (e.g. MySQL Workbench)

3. Create a .env file called `.env.local` instead of `.env` and file's credentials with your db credentials

4. Start Dev Server

   ```bash
   npm run dev
   ```

5. Compile project

   ```bash
   npm run build
   ```

6. Start Production Server

   ```bash
   npm run start
   ```
