# School-Administration-System-Backend-APi

## To start external api

1. Open terminal

- cd external

2. Install the required modules

```bash
npm install
```

3. Start server

```bash
npm run start
```

## To Start TypeScript Project

1. Open another terminal

   ```bash
   cd typescript
   ```

2. Install the required modules

   ```bash
   npm install
   ```

3. Use your MySQL Editor to create tables from `typescript/DDL.sql` (e.g. MySQL Workbench)

4. Update `.env.local` file's credentials with your db credentials

5. Start Dev Server

   ```bash
   npm run dev
   ```

6. Compile project

   ```bash
   npm run build
   ```

7. Start Production Server

   ```bash
   npm run start
   ```
